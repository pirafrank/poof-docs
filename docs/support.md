## Bug Reports

If you encounter any issues not covered above, please [report them](https://github.com/pirafrank/poof/issues/new/choose) by opening an issue.

**When reporting bugs, please include:**

- Output from `poof info`
- Debug logs: `RUST_LOG=debug poof <command>` output (see [Debug Logs](./debug-logs.md) for instructions)
- Operating system and architecture
- The specific command that failed
- Expected vs actual behavior

## Feature Requests

If you have an idea or an improvement to an existing feature to suggest, please [tell us](https://github.com/pirafrank/poof/issues/new/choose) by opening an issue of the proper type.

## Discussions

The best place to talk about topics before getting into the code is the [Discussions](https://github.com/pirafrank/poof/discussions) section.
